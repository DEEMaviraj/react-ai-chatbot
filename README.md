AI Chatbot (React + Node.js + Gemini API)
Project Overview

This is a simple AI-powered chatbot built using:

Frontend: React (Vite)
Backend: Node.js + Express
AI: Google Gemini API

The chatbot sends user messages to a backend server, which forwards them to the Gemini API and returns AI-generated responses.

Features:
Real-time chat interface
AI responses using Gemini
Auto scroll chat messages
Clean React component structure
Backend API with Express
Environment variable support (.env)

Tech Stack:
React (Vite)
Node.js
Express.js
Gemini API
JavaScript (ES6+)
CSS
